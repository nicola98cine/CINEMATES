IL contenuto delle 4 sottocartelle del repository CINEMATES e' il seguente
- Cinemates2020 è il progetto dell'app mobile , sviluppato con android Studio utilizzando Java e Gradle . Contiene le classi di Test Junit
- CinematesAdin è il progetto della componente Desktop , sviluppato con Eclipse, utilizzando Java JDK1.8 e framework Swing . Contiene le classi di TEst Junit
- ApiGateway  è l'SDK per Android fornito da AWS a seguito del deploy dell'API 
- LambdaFunctionOverHttps è il codice Node.js della funzione lambda sviluppata per effettuare tutte le operazioni CRUD utilizzate dalle applicazioni mobile e desktop
